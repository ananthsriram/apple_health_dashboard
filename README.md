# Apple Health Dashboard

A Flask-based web application to visualize and analyze your Apple Health workout data.

## Quick Start

1. **Install dependencies:**
   ```bash
   python3 -m venv venv
   venv/bin/pip install flask lxml tqdm
   ```

2. **Configure your data directory:**
   Edit `config.py` and set `DATA_DIR` to point to your Apple Health export folder:
   ```python
   DATA_DIR = '/path/to/your/apple_health_export'
   ```

3. **Process your data:**
   ```bash
   venv/bin/python process_data.py
   ```
   This will parse the `export.xml` file and generate CSV files and aggregated JSON files.

4. **Run the dashboard:**
   ```bash
   venv/bin/python app.py
   ```
   Open http://127.0.0.1:5000/ in your browser.

## Updating with New Data

When you export new Apple Health data:

1. **Option A: Update the config (Recommended)**
   - Export your Apple Health data to a new location
   - Edit `config.py` and update `DATA_DIR` to the new path
   - Run `venv/bin/python process_data.py`
   - Restart the app

2. **Option B: Replace in the same location**
   - Export your Apple Health data to the same folder specified in `config.py`
   - Run `venv/bin/python process_data.py` to regenerate the processed data
   - The app will automatically use the new data (no restart needed if using cache clearing)

## Features

- **Multiple Chart Types**: Line, Bar, Radar, Pie, Doughnut, Polar Area
- **Activity Filtering**: View individual activities or Total (all combined)
- **Granular Data**: All metadata and statistics from Apple Health are preserved in CSV files
- **Fast Performance**: Pre-aggregated JSON files for instant dashboard loading
